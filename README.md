## Project Description
This project is an interactive Tic Tac Toe web application developed using HTML, CSS, and JavaScript.  
The game allows a user to play against an AI opponent with a modern purple-themed interface, smooth animations, and real-time score tracking.

## Features
- Play against AI opponent
- Smart AI that blocks and attempts to win
- Purple neon themed UI
- Animated moves and winning highlight
- Score tracker for Player vs AI
- Reset board functionality

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the file `tic-tac-toe-ai.html`
2. Open it in any modern web browser
3. Start playing against the AI
